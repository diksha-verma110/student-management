# Student Management System 
### This repo contains a simple **Student Management System** built with **React.js** and **Vite**. This project demonstrates the basic frontend structure of a student portal where users can perform simple operations such as login, attendance tracking, timetable viewing, and result updates.
**Features**
---
 Parent Login - Simple login form for parents.
 Student Login - Allows students to log in with their roll number.  
 Attendance - Teachers can mark students present.  
 Timetable - Displays a static class timetable.  
 Result Update - Allows entering and displaying student marks.
